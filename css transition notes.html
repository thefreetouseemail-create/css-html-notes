🔥 CSS Transition Property

.box {
    width: 22px;
    height: 42px;
    background-color: rgb(239, 52, 205);
    border-radius: 10px;

    transition-property: transform;
    transition-duration: 3s;
    transition-timing-function: cubic-bezier(.23, .6, .46, .82);
    transition-delay: 0.3s;
}

.box:hover {
     transform: translate(58vw);
}

Transition Shorthand

transition: property duration timing-function delay;
Value मतलब
property कौन-सी property change होगी
duration change होने में कितना time लगेगा
timing-function speed pattern
delay कितने time बाद start होगा 
transition: transform 3s ease-in-out 0.3s;
