🔥 CSS Flexbox Notes

1️⃣ Flex Axis

Flexbox में दो axis होती हैं।

Axis क्या है
Main Axis जिस direction में items चलते हैं
Cross Axis Main axis के opposite direction

अगर

flex-direction: row;
तो 
Main Axis  → left to right
Cross Axis ↓ top to bottom

अगर

flex-direction: column;

तो

Main Axis  ↓ top to bottom
Cross Axis → left to right

2️⃣ Flex Direction

items किस direction में arrange होंगे।

flex-direction: row;
value क्या करेगा
row left → right
row-reverse right → left
column top → bottom
column-reverse bottom → top

3️⃣ Justify Content

Main Axis में items की alignment।

justify-content: center;
value क्या करेगा
flex-start start में
flex-end end में
center center
space-between items के बीच space
space-around हर item के around space
space-evenly equal space

4️⃣ Align Items

Cross Axis में items की alignment।

align-items: center;
value क्या करेगा
stretch default
flex-start start
flex-end end
center center
baseline text baseline

5️⃣ Flex Wrap

items next row में जाएंगे या नहीं।

flex-wrap: wrap;
value क्या करेगा
nowrap default, wrap नहीं
wrap next line में
wrap-reverse reverse direction wrap

6️⃣ Wrap Reverse

items next line में जाएंगे लेकिन reverse direction में।

flex-wrap: wrap-reverse;

मतलब नई row ऊपर की तरफ बनती है।

7️⃣ Align Content

multiple rows के बीच space control करता है।

⚠️ यह तभी काम करेगा जब:

flex-wrap: wrap;
align-content: center;
value क्या करेगा
flex-start ऊपर
flex-end नीचे
center center
space-between rows के बीच space
space-around rows के around space
stretch default

8️⃣ Flex Flow

flex-direction + flex-wrap का shortcut।

flex-flow: flex-direaction value (space) flex-wrap value; 
flex-flow: row wrap;

9️⃣ Gap Property

items के बीच distance।

gap: 20px;

या

row-gap: 20px;
column-gap: 10px;

या

gap: row column
gap: 20px 12px;

🔟 Item Property

ये properties flex items पर लगती हैं।

Order

items की position change करता है।

order: 2;

default

order: 0

जिसका order कम होगा वह पहले आएगा।

NOTE: "-" accending "+" descending

Flex Grow

extra space को fill करता है।

flex-grow: 1;

example

item1 → flex-grow:1
item2 → flex-grow:2

तो item2 double space लेगा।

Flex Shrink

space कम होने पर item कितना shrink होगा।

flex-shrink: 1;

Align Self

single item की alignment change करता है।

align-self: center;
align-content: none; hone jaruri hai.
value
flex-start
flex-end
center
stretch
