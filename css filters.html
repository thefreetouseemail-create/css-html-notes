🔥 CSS filter — Quick Notes (Simple but Useful)

काम:

filter CSS property से image / element पर visual effects लगाए जाते हैं — जैसे blur, brightness, grayscale आदि।

👉 Mostly use होता है:

  • images
  • icons
  • hover effects
  • UI effects

1️⃣ Basic Syntax

img{
  filter: blur(4px);
}

या multiple filters

img{
  filter: brightness(120%) contrast(110%);
}

📌 Multiple filters space से separate होते हैं।

2️⃣ Important Filter Functions

Filter काम
blur(px) image blur करता है
brightness(%) light / dark control
contrast(%) contrast increase या decrease
grayscale(%) black & white effect
sepia(%) brown vintage effect
saturate(%) colors strong करता है
hue-rotate(deg) colors change
invert(%) colors उल्टा
opacity(%) transparency
drop-shadow() shadow effect

ये सभी CSS filters standard हैं।
