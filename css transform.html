🔥 CSS Transform

1. Transform Property क्या है

transform CSS property किसी element की shape या position change करती है बिना document flow को disturb किए।

2. Transform Functions (Main Concepts)

(1) translate()

Element को move (shift) करता है।

translateX()

Element को left-right move करता है.

transform: translateX(50px);

translateY()

Element को up-down move करता है.

transform: translateY(50px);

translate()

transform: translate(50px, 20px);

पहली value → X direction
दूसरी value → Y direction

(2) scale()

Element का size increase या decrease करता है.

scaleX()

transform: scaleX(2);

width 2× हो जाएगी

scaleY()

transform: scaleY(2);

height 2× हो जाएगी

scale()

transform: scale(1.5);

Element 1.5 times बड़ा हो जाएगा

(3) rotate()

Element को rotate करता है.

transform: rotate(45deg);

rotateX()

rotateX 👆

rotateY()

rotateY 👆

rotateZ()

rotateZ 👆

(4) skew()

Element को tilt (तिरछा) करता है.

skewX()

transform: skewX(20deg);

skewY()

transform: skewY(20deg);

skew()

transform: skew(20deg,10deg);

3. Multiple Transform

एक element पर multiple transforms apply कर सकते हैं।

transform: translateX(100px) rotate(45deg) scale(1.2);

Order important होता है।

4. transform-origin

Element किस point से transform होगा यह define करता है।

Default:

center center

Example:

transform-origin: top left;
transform: rotate(45deg);

अब rotation top-left corner से होगा।

Values:

value मतलब
center middle
top ऊपर
bottom नीचे
left left
right right

5. Transform Combination

transform: translatey(-50px) rotate(-45deg) scaley(1.2) skewx(50deg);

6. Transform + Transition (Most Used Combo)

.box{
  transition: 0.3s;
}

.box:hover{
  transform: scale(1.1);
}

अब zoom smooth animation से होगा।
