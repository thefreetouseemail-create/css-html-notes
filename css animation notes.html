@keyframes spin {
    0% {}

    50% {
        background-color: yellow;
    }

    100% {
        transform: translatex(200px) rotate(360deg) scale(2.4);
        border-radius: 50%;
    }
}

    cursor: pointer;
    animation-name: spin;
    animation-duration: 3s;
    animation-timing-function: ease-in-out;
    animation-delay: 2s;
    animation-iteration-count: 4;
    animation-direction:alternate;
    /* animation-fill-mode: forwards; */

Property Function
duration Animation कितने time तक चलेगा
Property Function
timing-function Animation किस speed pattern से चलेगा
Value मतलब
linear same speed
ease slow start, fast middle
ease-in slow start
ease-out slow end
ease-in-out slow start + slow end
Property Function
delay Animation कितने time बाद start होगा
Property Function
iteration-count Animation कितनी बार repeat होगा
Value मतलब
number जितनी बार number दिया होगा
infinite animation लगातार चलता रहेगा
Property Function
direction Animation किस direction में चलेगा 
    📌 Note:
    alternate में 1 जाना + 1 आना = 2 iterations
Value मतलब
normal start → end
reverse end → start
alternate start → end → start
alternate-reverse end → start → end
Property Function
fill-mode Animation खत्म होने के बाद element किस state में रहेगा
Value मतलब
none default state
forwards last frame पर रुक जाएगा
backwards first frame apply
both first + last frame apply
