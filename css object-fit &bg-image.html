🔥 CSS object-fit (Image / Video Fit Control)

  • background-image

काम:
object-fit यह तय करता है कि image या video अपने container (box) के अंदर कैसे fit होगा।

📌 ज़्यादातर use होता है <img> और <video> के साथ। 

img{
  width:300px;
  height:200px;
  object-fit: cover;
}

⚠️ जरूरी बात:

object-fit तभी सही काम करता है जब width और height दी हुई हों।

1️⃣ object-fit: fill (Default)

Image को container के exact size में stretch कर देता है।

.body{
    background-color:rgb(124, 202, 202);
    border: 5px solid red;
    height: 300px;
    width: 300px;
}
img{
    height: 300px;
    width: 300px;
    object-fit: fill;
}

✅ container पूरा fill हो जाता है
❌ image stretch / distort हो सकती है


📌 Use case
  • लगभग use नहीं किया जाता
  • क्योंकि image खराब दिख सकती है

2️⃣ object-fit: contain

Image को पूरी दिखाता है, बिना crop किए।

img{
  object-fit: contain;
}

✅ image का aspect ratio maintain
✅ image cut नहीं होती
❌ container में खाली space आ सकता है


📌 Use case
  • product images
  • logos
  • जब पूरी image दिखानी जरूरी हो

3️⃣ object-fit: cover ⭐ (सबसे ज्यादा use)

Image container को पूरी तरह भर देती है।

img{
  object-fit: cover;
}

✅ container पूरा fill
✅ image distort नहीं होती
❌ image का कुछ हिस्सा crop हो सकता है


📌 Use case
  • cards
  • banners
  • thumbnails
  • profile pictures
👉 Real websites में सबसे ज्यादा यही use होता है

4️⃣ object-fit: none

Image original size में ही रहती है।

img{
  object-fit: none;
}

✅ image resize नहीं होती
❌ container से बाहर जा सकती है

Image = original size
Container = छोटा

📌 Use case
  • rarely use
  • custom cropping

5️⃣ object-fit: scale-down

यह contain और none में से जो छोटा हो वही apply करता है।

img{
  object-fit: scale-down;
}

✅ image unnecessarily बड़ी नहीं होती
✅ safe option


📌 Use case
  • responsive images
Value Image Cut Stretch Empty Space
fill ❌ ✅ ❌
contain ❌ ❌ ✅
cover ✅ ❌ ❌
none ❌ ❌ ❌
scale-down depends ❌ possible

🔥 object-position (Bonus Property)

यह decide करता है कि image का कौन सा हिस्सा दिखे।

img{
  object-fit: cover;
  object-position: top;
}

1️⃣ Single Direction Values

Example

object-position: center;
object-position: top;
object-position: bottom;
object-position: left;
object-position: right;

Example

img{
width:300px;
height:200px;
object-fit:cover;
object-position: top;
}

👉 इससे cropped image का focus area change कर सकते हैं।

2️⃣ Two Direction Values ⭐ (Most Useful)

यहाँ दो values दी जाती हैं
👉 horizontal + vertical

object-position: left top;
object-position: right top;
object-position: left bottom;
object-position: right bottom;
object-position: center center;
img{
  width:300px;
  height:200px;
  object-fit: cover;
  object-position: right top;
}

👉 image crop होगी लेकिन top-right area दिखेगा।

तुम percentage भी दे सकते हो।

object-position: 50% 50%;
object-position: 0% 0%;
object-position: 100% 0%;
object-position: horizontal vertical
Value Meaning
0% 0% top-left
50% 50% center
100% 0% top-right
0% 100% bottom-left
100% 100% bottom-right

🔥 CSS background-image

काम:

background-image से div या किसी element के पीछे image लगाई जाती है।

.box{
  background-image: url(image.jpg);
}

📌 Mostly use होता है:

  • hero sections
  • banners
  • cards
  • website backgrounds

1️⃣ background-image

Element के पीछे image लगाता है।

.box{
  background-image: url(image.jpg);
}

📌 Multiple images भी लगा सकते हैं

background-image: url(img1.png), url(img2.png);

2️⃣ background-size

Image का size control करता है।

background-size: cover;

Common values

Value Meaning
cover container पूरा fill
contain पूरी image दिखे
auto original size

Example

.box{
background-image: url(img.jpg);
background-size: cover;
}

📌 cover सबसे ज्यादा use होता है।

3️⃣ background-position

Image किस जगह से दिखेगी यह control करता है।

background-position: center;

Common values

background-position: top;
background-position: bottom;
background-position: left;
background-position: right;
background-position: center;

Two direction values

background-position: left top;
background-position: right top;
background-position: left bottom;
background-position: right bottom;
background-position: center center;

4️⃣ background-repeat

Image repeat होगी या नहीं यह control करता है।

background-repeat: no-repeat;
Value Meaning
repeat image repeat
no-repeat repeat नहीं
repeat-x horizontal repeat
repeat-y vertical repeat

5️⃣ background-clip

Background कहां तक दिखेगा यह decide करता है।

background-clip: border-box;
Value Meaning
border-box border तक
padding-box padding तक
content-box content area

⚡ Shortcut Property

इन सबको एक साथ भी लिख सकते हैं।

background: url(img.jpg) center/cover no-repeat;

Structure

background: url("image.jpg") image position / size repeat

Example

background: url(img.jpg) center/cover no-repeat;

🎯 object-fit vs background-image

Feature object-fit background-image
Use on <img> <div>
Image type real image element background
Control image fit background system
